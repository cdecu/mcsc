# mcsc
Yet Another MineCraft Server Controller<br>
for my kids

## Start and Init New Project
```bash
npm -g install typescript typings ...
mkdir mcsc
cd mcsc
npm init
tsc init
typings init 
npm install debug yargs --save
typings install  --global  dt~node dt~nconf --save
start coding
ezv
```
Replace nconf by yargs
